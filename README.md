# 📄 Resume Classification App

## 🔹 Project Description
This project is a Machine Learning-based web application that classifies resumes into different job categories such as React Developer, SQL Developer, Peoplesoft, Workday, and Internship roles.

## 🔹 Objective
To automate the resume screening process using NLP and Machine Learning techniques.

## 🔹 Features
- Upload resume (PDF/DOCX)
- Automatic text extraction
- Resume classification
- Prediction probability display
- Resume preview
- Skill detection and missing skills
- User-friendly Streamlit UI

## 🔹 Technologies Used
- Python
- Machine Learning (Naive Bayes)
- NLP (TF-IDF)
- Streamlit
- Pandas, NumPy

## 🔹 Workflow
1. Data collection
2. Data preprocessing
3. Feature extraction (TF-IDF)
4. Model training
5. Model evaluation
6. Deployment using Streamlit
